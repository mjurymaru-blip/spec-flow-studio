# Aether Console ロードマップ

## 概要

Aether ConsoleはSF風のAIオペレーションルームPWA。複数のAIエージェントを可視化・制御し、最終的にはSpec-Flow Studioを監視する。

## 開発方針

```
Aether Console (先行開発)
     ↓
Spec-Kit形式を設計・共通化
     ↓
Spec-Flow Studio (後続開発)
     ↓
両者連携で完成
```

---

## Phase 1: フロントエンド完成

**目標**: SF風司令室UIの完成、モックデータでの動作確認

| タスク | 状態 | 詳細 |
|--------|------|------|
| SvelteKitプロジェクト初期化 | ✅ | TypeScript, PWA設定済み |
| SF風デザインシステム | ✅ | global.css, Panel, GlitchText, StatusIndicator |
| 司令室レイアウト | ⬜ | 複数パネルのグリッドレイアウト |
| エージェントパネル | ⬜ | 各エージェントの状態表示 |
| コンソール出力表示 | ⬜ | ログ/メッセージのストリーム表示 |
| GitHub Pages デプロイ | ⬜ | adapter-static設定 |

**成果物**: 静的なSF風ダッシュボード（モックデータ）

## エージェント構成（予定）

| エージェント | 役割 | 監視対象 |
|-------------|------|---------|
| **Analyzer** | 情報解析 | 入力データの分析 |
| **Predictor** | 予測 | トレンド・リスク予測 |
| **Planner** | 計画立案 | 次のアクション提案 |
| **Monitor** | リスク監視 | 異常検知・アラート |

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | SvelteKit + TypeScript |
| スタイリング | Vanilla CSS |
| PWA | vite-plugin-pwa |
| デプロイ | GitHub Pages → Vercel（Phase 3） |
| AI API | Gemini / Claude（Phase 3） |
| 通信 | WebSocket / SSE（Phase 3） |
