# レビューフィードバック取り込みワークフロー

ChatGPT/Geminiからのレビュー結果を整理し、修正を実行します。

## 入力フォーマット（推奨）

ユーザーは以下の形式でフィードバックを貼り付けてください：

```markdown
## 外部AIレビュー結果

### 採用する指摘
- [ ] ○○の修正
- [ ] △△の追加

### 検討が必要な指摘
- □□について（理由: ...）

### 却下する指摘
- ××（却下理由: ...）
```

## 手順

1. レビュー結果の内容を解析し、対応項目を一覧化
2. 「採用する指摘」について、対象ファイルと修正内容を特定
3. 修正を実行
4. 「検討が必要な指摘」について、ユーザーと相談
5. 修正完了後、関連ドキュメント（docs/）も更新
6. 変更サマリーを報告
