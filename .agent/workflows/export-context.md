# コンテキストエクスポートワークフロー

repomixを使用してプロジェクト全体を1ファイルに書き出し、外部AIレビューに備えます。

## 前提条件

repomixがインストールされていること。
未インストールの場合は `npm install -g repomix` を実行。

## 手順

// turbo
1. `npx repomix --output context.txt` でプロジェクトをエクスポート
2. 生成された `context.txt` の内容サマリーを報告
3. ユーザーに「context.txt をGemini/ChatGPTに渡してレビュー依頼してください」と案内

## エクスポート後のレビュー依頼テンプレート

```
以下はプロジェクトのソースコード全体です。
以下の観点でレビューをお願いします：

1. 設計・アーキテクチャの整合性
2. セキュリティ上の懸念
3. UI/UXの改善案
4. パフォーマンス最適化の余地

---
[context.txtの内容をここに貼り付け]
```
